Understanding Bitwise Operators

HELLO_E$THER
2019-10-10
Algorithms
  • & (bitwise AND)
  • | (bitwise OR)
  • ~ (bitwise NOT)
  • ^ (bitwise XOR)
  • << (bitwise left shift)
  • >> (bitwise right shift)
  • >>> (bitwise unsigned right shift)
  • &= (bitwise AND assignment)
  • |= (bitwise OR assignment)
  • ^= (bitwise XOR assignment)
  • <<= (bitwise left shift and assignment)
  • >>= (bitwise right shift and assignment)
  • >>>= (bitwise unsigned right shift and assignment)

& 与

0 & 0 = 0
0 & 1 = 0
1 & 1 = 1

Usage: 奇偶检查

| 或

0 | 0 = 0
0 | 1 = 1
1 | 1 = 1

Usage: 条件判断

~ 非

~0 = 1
~1 = 0

Usage: 补数

^ 异或

0 ^ 0 = 0
0 ^ 1 = 1
1 ^ 1 = 0

<< 左移

a << b is shifting the number a to the left by b places, and add 0 to the last b position
a << b == a * Math.pow(2,b)

Usage: 乘以2的倍数

>> 右移

a >> b is shifting the number a to the right by b places, and add 0 to the last b position
a >> b == a / Math.pow(2,b)

>>> 无符号右移，逻辑右移

a >> b is shifting the number a to the right by b places, and add 0 to position.
