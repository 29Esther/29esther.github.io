搞笑片度

这个作者是有点子好笑在身上的。

他在序言部分介绍《Is the book for me？》里面写到，我们幽默感共通，因为你已经读到这里啦。到最后的时候，他说，你已经读到这里啦，说明这本书真的很适合你，因为很明显，你喜欢我，而且，你知道么？我也喜欢你。

然后，第一章《How to use this book》，他在里面写，如果你读的是纸质版的话【P.S.我读的就是纸质版，这本书有点厚的，和Java笔记差不多厚】，这本书是有点厚的，你可能会发现它很适合用来垫显示器，提高高度。233333…

第五节，在谈到各种编程语言的未来发展时，他觉得我们没必要也不可能预测某一个编程语言的发展和未来，除非你有水晶球，但是如果你有的话，别管什么编程语言啦，华尔街更适合你。

第十三节在讨论简历。作者的观点是找专人写简历，因为写简历是一门艺术，就像装修你会找团队，并不是自己做不了。他还调侃了CEO，说CEO可能会写网页，也会写程序，可是他不写，让你写，因为你是专业的，更因为，他想把时间花在更重要的事情上面，比如假装在工作和打高尔夫球。后面，他在介绍一个好的简历应该是什么样的时候，开头介绍说：不管你是请一个简体作家，还是自己写（嗯，*咳嗽*，不要这么做），你都应该谅解一个好的简历应该是什么样的。他写的这个咳嗽，好好笑，好像画外音，感觉就是看书的时候，作者也在旁边。然后，他建议简历要短一些，最好控制在一页。他说他自己的第一份简历六页长，他详细列举了他的每一份工作，工作中用到的每一个技术，获得的每一个证书，以及他在过去两年早饭吃了什么。六页长的简历没有人会读的，就算发给你妈妈，她也只会假装读了。

家人们，谁懂啊。看这么严肃的书，有的时候我真的感到很困，倒头就睡。但是，作者的每一包袱，我又觉得有点好笑的。在说面试的14章，他建议大家穿得正式一些，为了给面试官留下专业的好印象，至少要高于公司标准的1-2层。但是，他不会穿礼服，因为有点太过了，除非面试的是秘密特工职位。后面他建议大家提前10分钟到达面试场地。要刚刚好好提前10分钟。如果你不能/不习惯准时参加会议，那这边就是一个建议提前30分钟到，但是，花20分钟做一些其他的事情。（在没有人看到的地方，在办公楼外面，我不应该这么说，但是我说了，所以你看到了这句话）。
